Active Directory Lab Documentation

This document outlines all the steps, configurations, and commands used in the lab.

Step 1: Domain Controller Installation

Install-WindowsFeature AD-Domain-Services
Install-ADDSForest -DomainName "Lab.local"

Installed the first domain controller and promoted it to the forest root.

Step 2: Organizational Units & Users

New-ADOrganizationalUnit -Name "IT_Department" -Path "DC=Lab,DC=local"
New-ADUser -Name "John Doe" -AccountPassword (ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force) -Enabled $true

Created OUs and user accounts for testing group policies.

Step 3: Group Policies

New-GPO -Name "PasswordPolicy" -Comment "Enforces strong passwords"
Set-GPInheritance -Target "OU=IT_Department,DC=Lab,DC=local" -IsBlocked $false

Implemented password policies and applied them to relevant OUs.

Reflection & Lessons Learned

This lab reinforced my understanding of Active Directory architecture, OU design, user management, and Group Policy implementation. I gained hands-on experience troubleshooting domain issues and automating tasks with PowerShell.
